# PC Config — отдельные ассеты для лендинга

Все ассеты разложены отдельными файлами.

## Структура

- `hero/` — основной рендер ПК для первого экрана.
- `categories/` — отдельные изображения категорий комплектующих.
- `builds/` — отдельные изображения популярных сборок.
- `icons/` — отдельные SVG-иконки преимуществ и фич.
- `ui/` — отдельные SVG-элементы интерфейса: логотип, play, arrows, шаги, тема.

## Цвета

- Primary blue: `#0B6BFF`
- Soft blue: `#6EA2FF`
- Background: `#FFFFFF`
